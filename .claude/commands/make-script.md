output/$ARGUMENTS 폴더 안의 sources.json (또는 slides.json)을 먼저 읽고, 슬라이드 순서를 파악한 뒤 아래 규칙으로 릴스 대본을 작성해서 output/$ARGUMENTS/script.txt에 저장해줘.

## 슬라이드 순서 파악 규칙

- sources.json이 있으면 → slide_n 값 오름차순 기준
- slides.json이 있으면 → n 값 오름차순 기준
- JSON 없이 이미지만 있으면 → 파일명 slide-01, slide-02... 오름차순 기준
- 절대 이미지 업로드 순서나 화면 표시 순서로 읽지 않음

## 대본 구조

첫 줄 고정: 소아청소년꽈 전문의가 알려드립니다.
마지막 줄 고정: 소아꽈언니의 소아꽈수첩입니다.
슬라이드 순서대로 핵심 내용 한 문장씩

## 자수 기준

- 전체 목표 180~200자 (공백 제외)
- 슬라이드마다 자수 최대한 균일하게

## 말투

- 구어체, 말하듯 자연스럽게
- 문어체 금지 (~합니다 → ~해요, ~입니다 → ~이에요)

## 숫자 → 한글 변환

- 400 → 사백, 600 → 육백
- 38℃ → 삼십팔도, 26℃ → 이십육도
- 24시간 → 하루, 12개월 / 1세 → 돌
- 개월 단위는 붙여쓰기: 6개월 → 육개월, 9개월 → 구개월, 18개월 → 십팔개월, 30개월 → 삼십개월
- 분(分) 앞 숫자는 붙여쓰기: 5분 → 오분, 20분 → 이십분

## 영어 → 한글 발음 변환

- AAP → 에이에이피
- IU → 아이유
- SPF → 에스피에프
- DEET → 디트
- HEPA → 히파
- ORS → 경구수액제

## 전화번호

- 119 → 일일구에 전화하세요
- 112 → 일일이에 전화하세요

## 기호 제거

- · / % ℃ 등은 말로 풀어서

## 저장 후 출력

- 슬라이드별 자수 및 대본 한 줄씩
- 총 자수
- 예상 시간 (총자수÷7초 ~ 총자수÷6초)
