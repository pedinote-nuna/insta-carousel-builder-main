## 디자인 DNA — A + B 하이브리드 시스템

소아과언니 카드뉴스의 시리즈 일관성은 **strict 시각 룰** 이 아니라
**톤 · 컬러 · 폰트 · 시그니처 텍스트** 로 만들어진다.
이미지 모델(나노바나나)에게 시각 자율성을 충분히 주어 일러스트를 풍성하게
만드는 대신, 아래 4가지가 9장에 동일하면 인스타 피드에서 시리즈로 인식된다.

### 시리즈 일관성 4요소
1. **톤**: 모던 의학 에디토리얼 (또는 다른 등록된 톤)
2. **컬러 팔레트**: 팔레트 A 또는 B 중 하나 (한 캐러셀 = 한 팔레트)
3. **타이포**: Pretendard 같은 한글 sans-serif, bold headline,
   헤드라인 마지막 단어를 액센트 컬러로 강조
4. **시그니처 텍스트**: 매 슬라이드에 "소아과언니" 한글 작게 노출

레이아웃·일러스트·여백·아이콘 종류·좌측 바 유무 같은 시각 요소는
**모델 자율** 영역. 좌표·픽셀 단위 강제 룰은 사용하지 않는다.

### 공통 (양쪽 팔레트 동일)
- 해상도: 1080×1350 고정 (인스타 캐러셀 4:5)
- 폰트: Pretendard 같은 한글 sans-serif, bold headline / regular body
- 헤드라인 룰: 마지막 단어만 액센트 컬러로 강조
- 좌측 정렬, 큰 여백, 잡지 에디토리얼 톤

### 팔레트 A — Editorial Coral (기본, ~60%)
- 배경: #FFFFFF (또는 살짝 따뜻한 off-white)
- 메인 텍스트: #1A1F36 (deep navy)
- 메인 액센트: `#C44536` (coral) — 헤드라인 마지막 단어 강조
- 보조: #5B7C99 (muted gray)
- 적용: 성장발달, 일상 케어, 영양, 소아심장 일반 교육, 앱활용

### 팔레트 B — Medical Teal (진지, ~40%)
- 배경: #FFFFFF (또는 살짝 따뜻한 off-white)
- 메인 텍스트: #0E1B2C (deep navy)
- 메인 액센트: `#2C6E63` (medical teal) — 헤드라인 마지막 단어 강조
- 보조: #6B7B8C (muted gray)
- 적용: 예방접종, 응급 상황, 약물 안전, 위험 신호, 질환관리

### 팔레트 결정 로직
**"이 정보를 잘못 적용하면 아이 건강에 즉각적 위험이 있는가?"**
- YES → B (Medical Teal)
- NO → A (Editorial Coral)
- 애매하면 A (기본값)

### 주의 액센트 — Mustard `#B8860B` (팔레트 무관 공통)
- 사용 위치 한정: "역행/제동" 슬라이드 — 주의·하지 마세요·너무 이릅니다·금지·위험
- 수량 제한: 한 캐러셀 9장 중 **1장만 권장**
- 머스타드는 라벨·헤드라인 단어·아이콘에 한정 적용 (메인 액센트 대체 X)

## 시그니처 표시 (모든 카드뉴스 절대 규칙)

### 슬라이드 1~8 (본문) — 통일된 시그니처
- 텍스트: **"소아과언니"** 한글 4글자 텍스트만. **캐릭터·이모티콘·이모지·작은 별·도트 마크·스마일·캐릭터 아이콘 등 장식 일체 금지**. 텍스트 자체가 시그니처의 전부
- 위치: **우상단** (top-right corner) 고정, 편안한 여백
- 폰트: Pretendard SemiBold
- 크기: **28px** (본문 헤드라인 대비 약 25%)
- 컬러: navy `#1A1F36`
- 부가 텍스트(직책·핸들·앱 등) 본문 영역에 절대 추가 금지
- **픽셀 좌표·위치 표기 절대 금지**: "80px", "x=100" 같은 좌표 값을 글자로 그리지 말 것 (이는 레이아웃 지시일 뿐 콘텐츠가 아님)

### 슬라이드 9 (Outro / CTA)
- 우상단 시그니처는 슬라이드 1~8과 동일 적용 (28px, navy, 80px 여백)
- 본문 영역에 다음 정보를 잡지 에디토리얼 스타일로:
  - 헤드라인 (저장 유도)
  - 출처 박스 (Tier 1·2)
  - **"소아과언니 / 소아청소년과 전문의"** 큰 브랜딩 블록
  - **`@dr.soa_unnie`** 핸들 + 팔로우 안내
  - **소아과수첩 앱** 안내
- 이모지 절대 금지 (❤️ 📘 ↗️ 등)
- 작은 장식(✨ ✦ 등) 금지
- 정보 위계 명확: 헤드라인 > 출처 > 브랜드 > CTA

### 한국어 환각 방지 (절대 룰)
- 프롬프트에 명시되지 않은 한국어 단어/문구를 **절대 만들지 말 것**
- 한국어 번역이 불확실하면 영어 원문 그대로 두거나 생략 (환각 X)
- 디자인 명세("Pretendard 700", "Bold" 등)는 지시문이므로 글자로 그리지 말 것
- 출처 박스의 "출처" 라벨은 슬라이드당 정확히 1회만 — 중복 금지
- 빈 공간이 어색해 보여도 환각 단어로 채우지 말 것 (의학 신뢰도 훼손)

### 안티 패턴
- ❌ 한 캐러셀 9장 중 머스타드 슬라이드 2장 이상 (모던 의학 에디토리얼 톤 한정 — 손그림 노트북 톤은 머스타드가 형광펜 메인 액센트라 별개)
- ❌ 머스타드를 옅게(peach·pale orange) 변형 — 항상 saturated
- ❌ 본문 슬라이드(1~8)에 영문 핸들 노출
- ❌ "소아과언니" 한글 시그니처 누락된 슬라이드
- ❌ 모델에게 좌표·픽셀 단위 디자인 룰 강제 (자율성 침해 → 일러스트 단조로워짐)
- ❌ 9장의 시각 요소(좌측 바·footer·페이지 번호)를 동일하게 강제하는 후처리 — 폐기됨

## 톤 라이브러리

카드뉴스의 시각 톤은 한 가지로 고정하지 않는다. 주제·정보 성격에 맞는
톤을 골라 적용하되, 모든 톤은 공통적으로 **시그니처 표시 규칙**(우상단
"소아과언니" 4글자, 28px navy, 장식 금지)을 따른다.

### 톤 1 — 모던 의학 에디토리얼 (등록 완료 · production 검증)
- 어울리는 주제: 일반 의료 정보, 영양, 발달, 일상 케어
- 배경: warm off-white `#FAFAF7` 또는 white `#FFFFFF`
- 메인 텍스트: deep navy `#1A1F36`
- 메인 액센트: coral `#C44536` (헤드라인 마지막 단어)
- 주의 액센트: mustard `#B8860B` (역행/제동 슬라이드 1장 한정)
- 보조 텍스트: muted gray `#5B7C99`
- 폰트: Pretendard 같은 한글 sans-serif, bold headline
- 일러스트: 잡지 에디토리얼, 사진 또는 깔끔한 라인 아트
- 검증: 이유식 9장 (output/example/) production 통과

### 톤 2 — 손그림 노트북 (이번 도입 · 검증 진행 중)
- 어울리는 주제: 일상 케어, 친근한 가이드, 단계별 설명, 부모 매뉴얼
- 배경: 모눈종이 warm beige `#F5F0E5` 또는 `#F8F2E3`
  · 옅은 펜슬 grid 라인 (rgba 0.1) — 노트 페이지 느낌
- 메인 텍스트: deep navy `#1A1F36` 또는 black `#2C2C2A`
- 형광펜 액센트(메인): mustard `#D4A84A` — 헤드라인 강조어 highlighter
- 경고 액센트: coral `#C44536` — **의학 안전 경고에만 한정** (안전수면·SBS·Red Flag)
- 보조 텍스트: warm gray `#5F5E5A`
- 폰트: Pretendard Bold + 손글씨 느낌 강조
- 일러스트 스타일: 손그림 라인 스케치, 거친 검정 펜 라인, 채움 없음(line art only)
- 박스 스타일: 손그림 사각형 + 모서리 cross marks (architectural draft 톤)
- 무드: 부모 메모장, 친근, 따뜻, 학습 노트
- 시그니처: 우상단 "소아과언니" — 다른 톤과 동일 룰
- 안티 패턴: 머스타드 형광펜은 헤드라인 강조어에만, 박스 외곽선 변경 금지

### 톤 추가 가이드
새 톤을 도입할 때:
1. 어울리는 주제 카테고리 명시 (애매한 톤 X)
2. 5요소 명시: 배경 / 메인 텍스트 / 메인 액센트 / 경고 액센트 / 보조 텍스트
3. 일러스트 스타일 1줄 정의
4. 시그니처 표시 규칙은 모든 톤 공통 — 변경 금지
5. 1세트 production 검증 후 "검증 완료" 라벨 부여
