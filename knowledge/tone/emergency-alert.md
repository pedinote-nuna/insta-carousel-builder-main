# 경고·긴급 톤 (Emergency Alert)

## 정의
응급 상황에서 즉각 행동을 유도하는 톤.
어두운 배경 + 강렬한 레드/민트 대비로 긴박감 전달.
부모가 경련·응급 상황에서 즉시 꺼내볼 수 있는 구조.

## 적합한 토픽
- 경련·응급 판단이 필요한 주제 (열성경련, 아나필락시스 등)
- 절대 하지 말 것 / 해야 할 것 대비가 명확한 주제
- 119 신호가 포함된 응급 대처 주제

## 디자인 토큰
- 배경: 짙은 네이비 #0D1B2A
- 긴급 강조: bright red #E63946
- 안전/긍정: mint #2EC4B6
- 텍스트: 흰색 #FFFFFF
- 카드 배경: #1A2B3C
- 폰트: Pretendard 계열 한글 sans-serif

## 레이아웃 패턴
- 커버: 상단 red stripe + 헤드라인 + 일러스트 크게
- 금지 슬라이드: red ❌ 아이콘 + 텍스트 + red border
- 행동 슬라이드: mint ✓ 아이콘 + 텍스트 + mint border
- 119 슬라이드: full-width red header + 구급차 일러스트 + red glow 박스
- 안심 슬라이드: 아이콘 카드 3개 적층
- 아우트로: 다크 배경 유지 + mint CTA

## 렌더링 가드
- 시그니쳐 "소아과언니" — 따옴표 없이 4글자만
- 숫자+px 텍스트 카드에 표시 금지
- 각 슬라이드 아이콘/일러스트 1개 이상 필수
- 텍스트 크기 절제 — 여백 충분히

## 소아과수첩 앱 CTA 매칭
- 응급·경련·야간 증상 → 야간·달빛병원 바로 찾기
- 응급 후 진료 준비 → 진료 메모
