# 캐릭터 일러스트 톤 (Character Illustration)

## 정의
영아·유아 영양·발달·돌봄 정보를 따뜻하고 친근하게 전달하는 톤.
파스텔 배경 + 귀여운 캐릭터 일러스트 + 부드러운 색감으로 부모와 아이 모두에게 편안한 인상.
초보 부모가 부담 없이 읽을 수 있도록 설계.

## 적합한 토픽
- 이유식·수유 등 영아 영양 시작 단계
- 발달 이정표·놀이·일상 케어
- 정보가 무겁지 않고 따뜻한 느낌이 핵심인 주제
- 응급·약물 같은 정확성 절대주의 주제는 부적합 (긴장감 부족)

## 디자인 토큰
- 배경: pastel cream #FFF8F0
- 카드 베이스: light cream #FFF0E0
- 메인 텍스트: warm brown #3D2B1F
- 메인 액센트: teal green #2C7A5C
- 경고 액센트: coral #E05C3A (금지 슬라이드 한정)
- 폰트: Pretendard 계열 한글 sans-serif (시그니처 SemiBold)

## 레이아웃 패턴
- 포맷: 1080×1350 portrait, 외곽 margin 충분히
- 시그니처: 우상단 '소아과언니' warm brown — 따옴표 없이 4글자만
- 라벨: 좌상단 작은 라벨 (예: '시작 시기', '준비 신호', '12개월 전 금지')
- 헤드라인: 마지막 단어 teal green underline (금지 슬라이드는 coral)
- 본문: 카드 그리드 (2×2, 2×3) 또는 stack — 카드마다 캐릭터·음식·사물 일러스트 1개
- 시각 요소: 캐릭터·일러스트 매 슬라이드 크게 1개 이상 (텍스트 도배 금지)
- 색감: 파스텔 톤 유지, 강한 대비 자제
- Outro: cream source box → warm brown brand block → 핸들 + 앱 CTA

## 렌더링 가드
- 따옴표(' ' / " ") 카드 위에 그리지 말 것 — 시그니처는 4글자만
- 숫자+px 텍스트 카드 표시 금지
- 캐릭터 일러스트 비중 높음 — 텍스트 정확도 떨어질 위험 큼 → 핵심 수치·금지 표현은 단순 명료하게
- 텍스트만 가득 찬 구성 절대 금지

## 소아과수첩 앱 CTA 매칭
- 이유식·수유·영양 → 성장 기록 ("성장 기록으로 이유식 시기 체크")
- 발달 이정표 → 발달 체크리스트
- 놀이·일상 → 진료 메모
