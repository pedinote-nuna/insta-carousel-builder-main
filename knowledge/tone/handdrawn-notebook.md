# 손그림 노트북 톤 (Handdrawn Notebook)

## 정의
의학 정보를 따뜻하고 친근하게 전달하는 톤.
모눈종이 노트 + 손으로 그린 일러스트 + 형광펜 강조.
육아 초보 부모님들에게 "무섭지 않은 의학 정보" 느낌을 줌.

## 적합한 토픽
- 영아 돌봄 (산통, 수면, 수유 등)
- 일상 육아 팁
- 행동·발달 관련
- 무서운 이미지 없이 따뜻하게 다룰 수 있는 소아과 주제

## 디자인 토큰
- 배경: 모눈종이 베이지 #F5F0E5
- 메인 텍스트: deep navy #1A1F36
- 형광펜 액센트: mustard #D4A84A (헤드라인 강조어 하이라이트)
- 경고 액센트: coral #C44536 (의학 안전 경고 슬라이드만)
- 일러스트: 검정 펜 손그림 line art, 채색 없음
- 박스: 손그림 사각형 + 모서리 cross marks (×)
- 폰트: Pretendard 계열 한글 sans-serif

## 레이아웃 패턴
- 시그니쳐: 우상단 "소아과언니" (매 슬라이드 공통)
- 커버(slide-01): 손그림 박스 프레임, 대형 타이틀, 중앙 일러스트, 부제 카피
- 정보 슬라이드: STEP XX · 영문명 레이블 → 신호 XX 헤드라인(mustard 하이라이트) → 일러스트 → 하단 카피
- 경고 박스: coral 테두리 손그림 사각형, ⚠️ 아이콘, 굵은 경고문
- 아우트로(slide-09): 출처 박스 + 소아과언니 브랜드 블록 + 앱 CTA

## 렌더링 가드 (모델 실수 패턴 방지)
- 숫자+px 형태(40px, 48px 등)는 절대 카드 위에 글자로 표시 금지
- 헤드라인 띄어쓰기는 입력값 그대로 유지, 임의 분리 금지
- "소아과언니" 4글자만, 장식·뱃지·픽셀값 추가 금지
- 따옴표(" ")를 카드 위에 그리지 말 것

## 소아과수첩 앱 CTA 매칭 기준
토픽별로 가장 관련 있는 기능 1개 선택:
- 열·해열제 관련 → 해열제 계산기
- 야간 증상·응급·산통 → 야간·달빛병원 바로 찾기
- 병원 방문 준비 → 진료 메모
- 성장·발달 → 성장 기록
- 약 복용·처방 → 약봉투 스캔
