# 에디토리얼 모던 톤 (Editorial Modern)

## 정의
잡지 에디토리얼 스타일의 깔끔하고 세련된 의학 정보 전달 톤.
화이트 배경 + coral 또는 teal 액센트로 신뢰감과 전문성을 동시에.
소아과언니의 기본(~60%) 톤.

## 적합한 토픽
- 성장발달 (발달이정표, 키·체중, 수면)
- 일상 케어·영양
- 소아심장 일반 교육
- 앱 활용 안내
- 위험도가 낮은 일반 육아 정보

## 팔레트 A — Editorial Coral (기본)
- 배경: #FFFFFF (또는 살짝 따뜻한 off-white)
- 메인 텍스트: #1A1F36 (deep navy)
- 메인 액센트: #C44536 (coral) — 헤드라인 마지막 단어 강조
- 보조: #5B7C99 (muted gray)
- 폰트: Pretendard 계열 한글 sans-serif

## 팔레트 B — Medical Teal (진지한 토픽)
- 배경: #FFFFFF
- 메인 텍스트: #0E1B2C (deep navy)
- 메인 액센트: #2C6E63 (medical teal)
- 보조: #6B7B8C (muted gray)

## 팔레트 결정 로직
"이 정보를 잘못 적용하면 아이 건강에 즉각적 위험이 있는가?"
- YES → 팔레트 B (Medical Teal)
- NO  → 팔레트 A (Editorial Coral) ← 기본값

## 레이아웃 패턴
- 좌측 정렬, 큰 여백, 잡지 에디토리얼 구성
- 헤드라인 마지막 단어를 액센트 컬러로 강조
- 커버: 대형 타이틀 + 서브타이틀 + 심플 일러스트
- 본문: 잡지 에디토리얼 스타일 카드/섹션 구성
- 경고박스: coral 또는 teal 테두리 박스
- 아우트로: 출처박스 + 브랜드블록 + 앱 CTA

## 주의 액센트 — Mustard #B8860B
- 사용 위치: "역행/제동/통념 깨기" 슬라이드 1장만
- 수량 제한: 한 캐러셀 9장 중 최대 1장

## 렌더링 가드
- 시그니쳐 "소아과언니" — 우상단, 따옴표 없이 4글자만
- 숫자+px 텍스트 카드에 표시 금지
- 헤드라인 띄어쓰기 입력값 그대로 유지

## 소아과수첩 앱 CTA 매칭
- 성장·발달 → 성장 기록
- 이유식·영양 → 성장 기록
- 병원 방문 준비 → 진료 메모
- 약 복용·처방 → 약봉투 스캔
- 소아심장 → 진료 메모
